MagicMath

Данный проект является коллективным, и выполняется командой "Животные" состоящей из 4 студентов. Список команды указан в Отчете №2

Цель проекта - разработка веб-приложения, которое представляет любое положительное целое число как сумму трех палиндромов.

Этапы разработки:

  • 1. Создание веб-страницы проета с помощью языка разметки html и каскадной таблицы стилей css
  • 2. Используя скриптовый язык программирования JavaScript разработать функцию, которая представляет любое целое положительное число, как сумму 3-х палиндромов
